4. STOREFRONT SECTION (FROM DASHBOARD)

Contains these tabs:

1. Public View

Shows how customers see the store.

Vendor can publish or unpublish the storefront.


2. Preview

Same as public view but only visible to vendor.

“View as Customer” mode.


3. Customize

Vendor edits everything about the products;

A. Products

Add products

Edit products

Delete products

Manage visibility (publish/unpublish)


B. About

Items shown publicly:

Banner

Logo

Store name + verified badge

Ratings + total reviews

Short bio

Categories sold

Approx. location

Online/offline status

Contact vendor (chat)

Follow Store (optional)


Storefront tabs:

Products

About

Reviews


C. Reviews

Public review system:

Average rating

Recent reviews

Filter by rating

Completed order count
